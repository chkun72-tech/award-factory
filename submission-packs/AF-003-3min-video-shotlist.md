# AF-003｜3 分鐘 Demo 影片分鏡

目標：讓評審在 3 分鐘內明白「這不是聊天機器人，而是小生意的營運 agent」。

## 0:00–0:15 開場

畫面：

- 標題：Handyman Quote-to-Follow-up Autopilot
- 副標：From messy customer messages to quote drafts and follow-up tasks

旁白：

> Local service businesses lose leads because customer enquiries are messy and follow-up takes time.

## 0:15–0:35 問題

畫面：

- 顯示一段客戶訊息。

```text
Hi, my bathroom tap is leaking and I may also need a cabinet fixed.
Can someone come this week? I am in Parramatta.
```

旁白：

> This message looks simple, but the owner still needs to identify the job type, ask missing questions, prepare a quote draft, and remember to follow up.

## 0:35–1:20 Agent 執行

畫面：

- 點擊 Run Autopilot。
- 顯示 agent workflow steps。

卡片：

- Service type
- Location
- Urgency
- Missing information
- Next action

旁白：

> The agent reads the enquiry, classifies the service, extracts location and urgency, and identifies missing information before preparing any customer-facing draft.

## 1:20–2:00 商業輸出

畫面：

- Job scope card
- Quote draft card
- Customer reply draft card
- Follow-up task card

旁白：

> The output is practical: a structured job scope, a non-binding quote draft, a polite customer reply, and a follow-up task for the owner.

## 2:00–2:30 技術架構

畫面：

```text
Customer message
  -> Intake parser
  -> Service classifier
  -> Missing-info checker
  -> Quote draft generator
  -> Follow-up task creator
  -> Owner dashboard
```

旁白：

> The workflow is designed as an AI agent pipeline. Qwen can power extraction and draft generation, while Alibaba Cloud can store job records and agent logs.

## 2:30–2:50 安全邊界

畫面：

- Badge: Human approval required
- Badge: Draft only
- Badge: No automatic customer sending

旁白：

> The system does not send messages automatically. It keeps the business owner in control.

## 2:50–3:00 結尾

畫面：

- Before: messy enquiry
- After: job scope, quote draft, reply, follow-up

旁白：

> This turns everyday admin work into an owner-ready workflow, helping small service businesses respond faster and follow up reliably.

## 錄製前 checklist

- Demo 可以從輸入跑到四個輸出。
- 畫面字體夠大。
- 不展示真實客戶資料。
- 不承諾正式報價。
- 影片控制在 2:45–3:00。
